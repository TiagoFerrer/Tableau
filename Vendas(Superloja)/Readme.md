# Análise de Vendas no Tableau
##### Este repositório contém um projeto de análise de dados na área de Vendas utilizando o Tableau. O objetivo deste projeto é analisar as vendas, desempenho dos produtos e indicadores de lucratividade para ajudar na tomada de decisões estratégicas.

### Pré-requisitos
##### Tableau Public

## Como utilizar
1. Baixe o projeto desejado no seu computador 
2. Abra o Tableau Public.
3. Navegue até a pasta do projeto desejado.
4. Abra o arquivo .twbx do projeto no Tableau.
5. Explore os dados, visualize os relatórios e painéis disponíveis no projeto.
